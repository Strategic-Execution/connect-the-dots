Organisational Change Analysis

📖 What is this tool?

Connect the Dots compares two weekly employee listing snapshots and automatically identifies every organisational change between them.

📥 Step 1 — Download Your Files

Go to the Group People, Performance & Reputation SharePoint site and download two W001-Employee-Listing CSV files.

📂 Open W001-Employee-Listing on SharePoint →

📊 Step 2 — Upload & Compare

  1. Upload the older file as the Old File (Baseline)
  2. Upload the newer file as the New File (Current)
  3. Click 🔍 Compare Files
